# Руководство по иконкам приложения (простое)

## Обзор

Этот скрипт только изменяет размер оригинального логотипа, сохраняя его дизайн без изменений.

## Файлы

### Исходный логотип
- `src/assets/logo.png` - оригинальный логотип (не изменяется)

### Скрипты
- `generate-icons-resize-only.sh` - генерирует иконки изменяя только размер логотипа
- `install-icons.sh` - устанавливает иконки в правильные директории

### Сгенерированные иконки
- `generated-icons/` - временная директория со сгенерированными иконками

## Использование

### Генерация иконок
```bash
./generate-icons-resize-only.sh
```

### Установка иконок
```bash
./install-icons.sh
```

### Полное обновление
```bash
./generate-icons-resize-only.sh && ./install-icons.sh
```

## Особенности

- ✅ Сохраняет оригинальный дизайн логотипа
- ✅ Только изменяет размер, без искажений
- ✅ Поддерживает все платформы (iOS, Android, Web)
- ✅ Сохраняет пропорции и качество

## Результат

Иконки выглядят точно как оригинальный логотип, только в нужных размерах для каждой платформы.
