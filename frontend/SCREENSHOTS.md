# Создание скриншотов для Google Play

## Обзор

Для вашего приложения "PL Price" созданы два скрипта для автоматического создания скриншотов через Android эмулятор.

## Скрипты

### 1. Интерактивный скрипт: `capture-screenshots.sh`
Позволяет вручную контролировать процесс создания скриншотов.

### 2. Автоматический скрипт: `auto-screenshots.sh`
Создает скриншоты основных экранов автоматически.

## Использование

### Запуск интерактивного скрипта:
```bash
./capture-screenshots.sh
```

### Запуск автоматического скрипта:
```bash
./auto-screenshots.sh
```

### Запуск с указанием другого эмулятора:
```bash
./auto-screenshots.sh My_Custom_AVD
```

## Требования к скриншотам Google Play

- **Минимум**: 2 скриншота
- **Максимум**: 8 скриншотов
- **Форматы**: JPG или PNG
- **Размеры**: 
  - Телефон: 320-3840px (минимальная сторона), максимальная сторона не более 7680px
  - Таблет: 600-7680px (минимальная сторона), максимальная сторона не более 7680px

## Процесс работы

### Автоматический режим:
1. Скрипт запускает эмулятор Android
2. Устанавливает ваше приложение
3. Создает скриншоты основных экранов:
   - Главный экран
   - Меню
   - Навигационная панель
   - Настройки
   - Различные состояния приложения

### Интерактивный режим:
1. Скрипт запускает эмулятор и приложение
2. Вы вручную управляете приложением
3. Нажимаете Enter для создания скриншота
4. Вводите команды для навигации

## После создания скриншотов

1. Просмотрите созданные файлы в папке `screenshots/`
2. Выберите лучшие 2-8 скриншотов
3. При необходимости отредактируйте их (добавьте описания, обрежьте)
4. Загрузите в Google Play Console

## Устранение проблем

### Если эмулятор не запускается:
```bash
# Проверьте доступные эмуляторы
$HOME/Library/Android/sdk/emulator/emulator -list-avds

# Запустите вручную
$HOME/Library/Android/sdk/emulator/emulator -avd Medium_Phone_API_36.1
```

### Если ADB не найден:
```bash
# Проверьте путь к Android SDK
echo $ANDROID_SDK_ROOT

# Установите путь если нужно
export ANDROID_SDK_ROOT=$HOME/Library/Android/sdk
```

### Если скриншоты пустые:
- Убедитесь что приложение полностью загрузилось
- Попробуйте увеличить время ожидания в скрипте
- Проверьте что эмулятор разблокирован

## Дополнительные возможности

### Создание скриншотов для разных размеров экранов:
```bash
# Создайте AVD с разными размерами
# Запустите скрипт для каждого AVD
./auto-screenshots.sh Tablet_API_36
./auto-screenshots.sh Large_Phone_API_36
```
