# Руководство по управлению иконками приложения

## Обзор

Ваши иконки приложения были сгенерированы на основе основного логотипа со звездами (`src/assets/logo-with-stars.png`) с сохранением правильных пропорций.

## Файлы

### Исходные логотипы
- `src/assets/logo.png` - основной логотип (польский флаг без звезд)
- `src/assets/logo-with-stars.png` - логотип с золотыми звездами в стиле ЕС над флагом

### Скрипты
- `create-logo-with-stars.sh` - создает логотип со звездами из базового логотипа
- `generate-app-icons.sh` - генерирует все необходимые размеры иконок из логотипа со звездами
- `install-icons.sh` - устанавливает сгенерированные иконки в правильные директории

### Сгенерированные иконки
- `generated-icons/` - временная директория со всеми сгенерированными иконками

## Использование

### 1. Создание логотипа со звездами
Если у вас есть только базовый логотип, создайте версию со звездами:
```bash
./create-logo-with-stars.sh
```

### 2. Генерация новых иконок
Если вы изменили логотип со звездами, перегенерируйте иконки:
```bash
./generate-app-icons.sh
```

### 3. Установка иконок
После генерации установите иконки в проект:
```bash
./install-icons.sh
```

### 4. Полное обновление
Для полного обновления всех иконок:
```bash
./create-logo-with-stars.sh && ./generate-app-icons.sh && ./install-icons.sh
```

## Типы иконок

### Web иконки
- `public/favicon.png` - основная иконка для браузеров (со звездами)
- `public/favicon.ico` - иконка для IE/Edge
- `public/favicon-*.png` - иконки разных размеров для PWA

### iOS иконки
- `ios/App/App/Assets.xcassets/AppIcon.appiconset/` - все размеры иконок для iOS (со звездами)
- Поддерживаются все необходимые размеры для iPhone и iPad

### Android иконки
- `android/app/src/main/res/mipmap-*/ic_launcher.png` - обычные иконки запуска (со звездами)
- `android/app/src/main/res/drawable-*/ic_launcher_foreground.png` - передний план адаптивных иконок
- `android/app/src/main/res/drawable-*-v26/ic_launcher_foreground.png` - адаптивные иконки (Android 8.0+)

## Особенности дизайна

### Звезды в стиле ЕС
- 5 золотых звезд расположены над польским флагом
- Звезды создают ассоциацию с Европейским Союзом
- Сохраняются пропорции оригинального логотипа
- Звезды видны на всех размерах иконок

### Сохранение пропорций
- Все иконки генерируются с сохранением пропорций логотипа со звездами
- Используется центрирование с прозрачным фоном для квадратных иконок
- Для Android создаются круглые иконки с синим фоном (#1e40af)

### Качество
- ImageMagick обеспечивает высокое качество при изменении размеров
- Все иконки оптимизированы для своих целевых размеров
- Звезды остаются видимыми даже на маленьких иконках

## Требования

- ImageMagick (устанавливается автоматически через Homebrew на macOS)
- Bash shell

## Устранение проблем

### Если ImageMagick не установлен
На macOS скрипт автоматически установит ImageMagick через Homebrew.

### Если иконки не обновились
1. Убедитесь, что вы создали логотип со звездами и перегенерировали иконки
2. Очистите кэш приложения:
   - iOS: Clean Build Folder в Xcode
   - Android: Clean Project в Android Studio
   - Web: очистите кэш браузера или перезапустите dev server

### Если звезды не видны на маленьких иконках
Убедитесь, что оригинальный логотип имеет достаточное разрешение. Можно изменить размер звезд в скрипте `create-logo-with-stars.sh`.

### Изменение цвета фона Android
Чтобы изменить цвет фона круглых иконок Android, отредактируйте строку в `generate-app-icons.sh`:
```bash
convert -size "${size}x${size}" xc:"#1e40af" ...
```
Замените `#1e40af` на нужный цвет.

### Изменение количества или расположения звезд
Отредактируйте массив `star_positions` в `create-logo-with-stars.sh`:
```bash
star_positions=("50,25" "100,15" "150,15" "200,15" "250,25")
```

## Автоматизация

Для автоматического обновления иконок при изменении логотипа можно добавить хук в Git:
```bash
echo "./create-logo-with-stars.sh && ./generate-app-icons.sh && ./install-icons.sh" > .git/hooks/pre-commit
chmod +x .git/hooks/pre-commit
```
